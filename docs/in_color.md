# Coloring strings

## `in_{color}` functions

Use the `in_{color}` functions to change the text color of a string

### in_red
```{eval-rst}
.. autofunction:: terminology.in_red
```
```{code-block} python
---
caption: Usage
---

in_red("Hello world!")
```

```{code-block} red
---
caption: Output
---
Hello world!
```
